# Подготовка проекта

Для начала нужно создать файлы переменных окружения для самого проекта, а также сервисов PostgreSQL и RebbitMQ 

```bash
.env
.env.postgres
.env.rabbitmq
```

## Генерирование ключей для JWT

Ключи хранятся в директории `creds/`. Необходимо перейти в неё и воспользоваться командами ниже.

```bash
openssl genrsa -out jwt-private.pem 2048
openssl rsa -in jwt-private.pem -outform PEM -pubout -out jwt-public.pem
```

## Настройка БД

Для подготовки БД также нужно провести миграции с помощью `alembic`. Перед этим важно указать данные, соответствующие развёрнутой БД в контейнере. Для начала вообще можно запустить только PostgreSQL:

```bash
docker compose up postgres
```

Далее в параметрах `.env` указать данные для подключения к БД с соответствующими портами из compose файла. После данной процедуры можно провести миграции.

```
poetry run alembic upgrade head
```

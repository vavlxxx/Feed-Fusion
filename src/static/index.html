Параметры запуска

Справка

  • epochs — сколько раз модель полностью проходит по всей обучающей выборке.
  • batch_size — сколько примеров модель обрабатывает за один шаг.
  • lr — скорость обучения (learning rate), определяющая силу изменения весов.
  • val_split — доля данных, откладываемая для проверки качества модели.
  • embed_dim — размер внутреннего векторного представления текста.
  • dropout — доля случайно отключаемых нейронов для борьбы с переобучением.
  • balance — включает компенсацию перекоса категорий в данных.
